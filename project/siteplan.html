Site Name

Scottish Castle Explorer

This name represents a tourism company that helps castle enthusiasts learn about the most famous castles in Scotland.

Site Purpose

This site provides the user with information about the most famous castles in Scotland and the opportunity to make a simple travel plan, as well as select their favorite castles from those presented.
